漢明距離

兩個整數之間的漢明距離指的是這兩個數字對應二進制位不同的位置的數目。

給出兩個整數 x 和 y，計算它們之間的漢明距離。

輸入: x = 1, y = 4

輸出: 2

解釋:
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑

上面的箭頭指出了對應二進制位不同的位置。

      
  /**
   * @param {number} x
   * @param {number} y
   * @return {number}
   */
  var hammingDistance = function(x, y) {
      var d1,d2
      var count=0
      do{
          d1=x%2
          d2=y%2
          count+=d1==d2?0:1
          x=(x-d1)/2
          y=(y-d2)/2
      }while(x>0||y>0)
      return count
  };
