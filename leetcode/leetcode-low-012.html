反轉字符串

編寫一個函數，其作用是將輸入的字符串反轉過來。輸入字符串以字符數組 char[] 的形式給出。

不要給另外的數組分配額外的空間，你必須原地修改輸入數組、使用 O(1) 的額外空間解決這一問題。

你可以假設數組中的所有字符都是 ASCII 碼表中的可打印字符。

輸入：["h","e","l","l","o"]
輸出：["o","l","l","e","h"]

      
  /**
   * @param {string} s
   * @return {string}
   */
  const reverseString = s => {
    let start = 0
    let end = s.length - 1
    while (start < end) {
      const item = s[start]
      s[start] = s[end]
      s[end] = item
      start++
      end--
    }
    return s
  }
