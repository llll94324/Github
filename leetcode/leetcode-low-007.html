加一

給定一個由整數組成的非空數組所表示的非負整數，在該數的基礎上加一。

最高位數字存放在數組的首位， 數組中每個元素只存儲單個數字。

你可以假設除了整數 0 之外，這個整數不會以零開頭。

輸入: [1,2,3]
輸出: [1,2,4]
解釋: 輸入數組表示數字 123。

      
  /**
   * @param {number[]} digits
   * @return {number[]}
   */
  var plusOne = function (digits) {
      for (let i = digits.length - 1; i >= 0; i--) {
          if (digits[i] < 9) {
              digits[i]++
              break
          } else {
              digits[i] = 0
          }
      }
      if (digits[0] == 0) {
          digits.unshift(1)
      }
      return digits
  };
