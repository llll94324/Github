爬樓梯

假設你正在爬樓梯。需要 n 階你才能到達樓頂。

每次你可以爬 1 或 2 個台階。你有多少種不同的方法可以爬到樓頂呢？

輸入： 3
輸出： 3
解釋： 有三種方法可以爬到樓頂。
1.  1 階 + 1 階 + 1 階
2.  1 階 + 2 階
3.  2 階 + 1 階

      
  /**
   * @param {number} n
   * @return {number}
   */
  var climbStairs = function(n) {
        let a = 1;
        let b = 1;
      
        for (let i = 0; i < n; i++) {
          let temp = a;
          a += b;
          b = temp;
        }

        return b;
  };
