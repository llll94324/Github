存在重復

給定一個整數數組，判斷是否存在重復元素。

如果任何值在數組中出現至少兩次，函數返回 true。如果數組中每個元素都不相同，則返回 false。

輸入: [1,2,3,1]
輸出: true
      
  /**
   * @param {number[]} nums
   * @return {boolean}
   */
  var containsDuplicate = function(nums) {
      var arr = [];
      for(var i = 0;i < nums.length;i++){
          if(arr[nums[i]]){
             return true
          }else{
              arr[nums[i]] = true;
          }
      }
      return false
  };
