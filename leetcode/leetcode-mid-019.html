子集

給定一組不含重復元素的整數數組 nums，返回該數組所有可能的子集（冪集）。

輸入: nums = [1,2,3]
輸出:
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
      
  /**
   * @param {number[]} nums
   * @return {number[][]}
   */
  var subsets = function(nums, res = [[]], index = 0) {
    if (nums.length <= index) return res
    return subsets(nums, res.concat(res.map(vv => vv.concat([nums[index]]))), index + 1)
  };
