實現 strStr()

給定一個 haystack 字符串和一個 needle 字符串，在 haystack 字符串中找出 needle 字符串出現的第一個位置 (從0開始)。如果不存在，則返回 -1。

輸入: haystack = "hello", needle = "ll"
輸出: 2

      
  /**
   * @param {string} haystack
   * @param {string} needle
   * @return {number}
   */
  var strStr = function(haystack, needle) {
      return haystack.indexOf(needle)
  };
